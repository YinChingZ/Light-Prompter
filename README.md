# Light-Prompter（单文件版）
这是一个基于纯前端实现的「轻量级提词器」。当前仓库提供单文件版本，便于分发与离线使用。

## 使用方式
- 文件：`single.html`（项目根目录）。
- 打开：直接双击或用浏览器打开。
- 操作流程：
  - 在“输入文本”框填入内容；可用标签 `[暂停:2]` 指示在中心线停顿 2 秒（支持小数）。
  - 调整滚动速度、字体大小、显示区域宽度/高度、水平位置后点击“开始提词”。
  - 可随时“暂停/继续”或“重置”，支持“全屏模式”。

## 功能概览
- 文本解析：支持 `[暂停:秒数]` 自动插入暂停点。
- 滚动动画：速度可调，帧率补偿更平滑。
- 版式控制：字体大小、区域宽度/高度、水平偏移可调，运行中保持相对位置。
- 全屏演示：全屏下有浮层控制面板，参数同步。
- 响应式：在移动端或小屏设备上自适应显示。

## 技术栈
- 原生 JavaScript、HTML5、CSS3；无第三方依赖。

该项目轻量、零依赖，适合作为网页提词器、视频直播、演讲演示等场景的前端工具。
